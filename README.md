# Projeto Escola

Site para Markting e divulgação de cursos, direção, eventos, noticias, upload de documentos etc... da escola com Angular + Spring Boot
 
## Tecnologias
- Frontend: Angular
- Backend: Spring Boot
- Banco: PostgreSQL

## Como executar
- Backend: `cd backend && ./mvnw spring-boot:run`
- Frontend: `cd frontend && npm start`
